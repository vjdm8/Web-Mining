This script is a KNN algorithm created from scratch.
It utilizes a created distance function to work.

The program requires the following libraries:

os
operator
collections
sklearn

Further the negative-words.txt, positive-words.txt, reviews_train.txt and reviews_test.txtn files 
are required in the same folder specified in the code path.

!!!!!!!!!!!!!!!!!!!!!!!!!!IMPORTANT!!!!!!!!!!!!!!!!!!!!!
Please adjust the path in the code to your working directory.
