A script that classifies textual data with sklearn
using multiple fine tuned classifiers, to find the 
most accurate classifier.

The program requires the following libraries:

os

sklearn

Further the reviews_train.txt and reviews_test.txtn files 
are required in the same folder specified in the code path.

!!!!!!!!!!!!!!!!!!!!!!!!!!IMPORTANT!!!!!!!!!!!!!!!!!!!!!
Please adjust the path in the code to your working directory.
